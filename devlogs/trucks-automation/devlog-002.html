🎮 Player Update

Hey everyone — quick update on Trucks & Automation and what’s been happening behind the scenes this week. We’re still in early alpha development, so this is all internal progress for now — but things are starting to come together in a big way.

⸻ Advanced Road Systems (Curves & Merges)

The road system has had a major upgrade. This is a huge step toward building proper, large-scale logistics networks.

  • Curved roads now allow smooth cornering.
  • Merge lanes let multiple routes combine into one.

⸻ Smart Junction Logic

Junctions are no longer “random” — they’re now controllable. This opens the door for fully custom logistics routing and planning.

  • New Junction Inspector system.
  • Ability to route trucks based on their identity.

⸻ Global Fleet Economy

Truck purchasing is now part of a shared system:

  • Buying more trucks increases the overall cost.
  • Encourages smarter expansion and planning.

⸻ Truck Inventory System

Trucks are now fully data-driven. This connects producers, storage, and transport into a real system.

  • Multi-resource inventories.
  • Capacity limits.
  • Proper loading/unloading behaviour.

⸻ UI Improvements

UI has been reworked to support the new systems:

  • Context-based inspectors for buildings and trucks.
  • Cleaner layout and readability.
  • Color coding for easier tracking.

⸻ What This Means

The core systems are now starting to link together properly: Roads → Trucks → Resources → Economy. This is a big step toward the full automation gameplay loop.

⚙️ Technical Dev Log

Focus: Core navigation stability, expanded tile logic, and system-wide data consistency

⸻ Physics & Navigation Overhaul

  • Conflict Resolution: Resolved rotation conflicts between grid alignment and trigger-based turning.
  • Separated Responsibilities: Grid system handles straight alignment, while triggers handle curves & junction logic.
  • Mathematical Grid Snapping: Ensures precise 90° turns and eliminates floating-point drift.
  • Cooldowns: Added turn cooldown system (0.4s) to prevent double-triggering in dense traffic.

⸻ Expanded Road Logic

  • New Tile Types: RoadLeft, RoadRight (curved nodes), and MergeNode (multi-lane funnel system).
  • Merge Nodes: Now override rotation using global orientation and allow clean multi-lane convergence.
  • Junction System Upgraded: Supports rule-based routing (truck-specific logic).

⸻ Global Economy Refactor

  • State Migration: Migrated garage data to static/global state.
  • Unified Economy: Shared truck count and shared pricing system ensures all garages operate as part of a unified economy.

⸻ Inventory System Stability

  • Initialization: Fixed initialization issues by preloading all enum values.
  • Error Handling: Prevented runtime dictionary errors (KeyNotFound).

⸻ Data-Driven Logistics

  • Data Structures: Implemented Dictionary<ResourceData, int> for truck inventory.
  • Scaling: Dynamic capacity scaling (base + level system).
  • UI Connection: Connected systems to UI for real-time updates for inventory, pricing, and coordinates.
