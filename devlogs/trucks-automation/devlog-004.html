Alpha v0.0004 22/04/2026 - 20:59PM

Smarter Trucks, Better Flow & Clearer Feedback

King Skyline [DHC]

🎮 Player Update

Big progress today on Trucks & Automation — the focus was making everything smarter, smoother, and easier to understand.

⸻ Trucks Are WAY Smarter

We tested with 100+ trucks at once, and the chaos is gone!

  • Trucks now look ahead before moving.
  • They won’t drive into missing roads or the wrong direction anymore.
  • They detect each other properly and form actual traffic queues instead of clipping through each other.

⸻ Traffic Problems = Visible Now

You can instantly see where your factory is struggling:

  • Trucks that sit too long now show a ⚠️ “stuck” warning.
  • Buildings that get jammed show a ⚠️ warning above them.

⸻ Logistics Fix (BIG ONE)

We fixed a major issue where trucks would clog the system. No more supply chain deadlocks!

  • Before: One resource could block everything.
  • Now: Trucks can carry multiple resources separately.

⸻ “Matrix Mode” (Debug Vision 👁️)

Hold Left Alt and you’ll see floating real-time data above the world. Super useful for understanding what’s going on:

  • Truck inventories
  • Building storage

⸻ Smarter Systems & Saves

  • Garages won’t spawn trucks if the exit is blocked.
  • The game now safely handles older save files.
  • Trucks keep their custom colours upon loading.

⸻ What This Means

The game now handles large traffic properly, shows you problems clearly, and runs more reliably. This is a big step toward a smooth, playable experience.

⚙️ Technical Dev Log

Focus: AI stability, traffic handling, logistics scalability, and player feedback systems

⸻ AI & Traffic Systems

  • Forward Grid Scanning: Implemented to prevent invalid movement (missing roads / wrong direction).
  • Front Radar: Added SphereCast-based front radar for vehicle detection and queuing. Stress tested with 100+ trucks to validate traffic flow.
  • Stuck Detection: Introduced a stuck detection timer. Trucks flag as “stuck” after prolonged idle time.
  • Spawn Validation: Added physics-based exit checks to garages to prevent blocked vehicle spawning.

⸻ Logistics System Upgrade

  • Slot-Based Capacity: Replaced global capacity with a slot-based capacity system.
  • Supports multi-resource transport without blocking, preventing cross-resource deadlocks.

⸻ Visual Feedback & Status Systems

  • Building Evaluation: Introduced BuildingStatus enum allowing buildings to self-evaluate operational state.
  • World-Space UI Warnings: Implemented visual indicators for blocked production and stalled trucks.
  • Data Overlay System: Built a toggleable (Alt key) real-time inventory display featuring camera-facing billboarding.

⸻ Stability & Persistence

  • Save Versioning: Added saveVersion system to ensure backward compatibility with older save files.
  • Extended Serialization: Save files now successfully include truck color data and full inventory states.
  • Improved UI Safety: Added robust null-checks to prevent runtime crashes from missing references.
