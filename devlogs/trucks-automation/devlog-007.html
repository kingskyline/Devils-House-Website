🎮 Player Update

We’ve just made a BIG change to how building works — and it makes the whole game feel more like a real factory sim.

⸻ Buildings Got an Upgrade

We’ve moved away from everything being tiny 1×1 tiles. Now, everything has a real sense of scale:

  • Producers are bigger machines.
  • Warehouses actually feel like warehouses.
  • HQ stands out as a proper main building.

⸻ Smarter Road Connections

  • Bigger buildings now have dedicated connection points.
  • Trucks know exactly where to go.
  • No more weird interactions — everything flows cleaner.

⸻ Why This Matters

Before: Everything felt the same size, making it easy to just spam buildings everywhere.

Now: Space matters. You actually have to plan your layout, making your factory feel much more real.

⸻ The Big Improvement

  • Better readability
  • More strategy
  • Cleaner layouts

It’s a small change technically, but a HUGE gameplay upgrade. More improvements coming soon as we keep scaling things up!

⚙️ Technical Dev Log

Focus: Grid scaling, multi-tile architecture, and gameplay readability

⸻ Multi-Tile Building System

Transitioned from 1×1-only buildings to a multi-tile footprint system with standardized building sizes:

  • Producers → 2×2
  • Warehouses → 3×3 / 4×4
  • Garages → 2×2
  • Selling Stations → 2×2 / 3×2
  • Headquarters → 4×4+
  • Logic Tools (routers/junctions) remain 1×1

GridManager updated to support:

  • Multi-key occupancy
  • Consistent placement validation
  • Clean demolition across full footprints

⸻ Road Connection Architecture

Implemented designated connection tiles for all multi-tile buildings and defined clear input/output nodes for logistics interaction. Resulting in:

  • Deterministic truck routing
  • Consistent snapping behaviour
  • A scalable building interaction system

⸻ Gameplay Structure Improvements

Introduced an intentional scale hierarchy (small tiles = precision tools, large buildings = strategic placement). Reduced “tile spam” by enforcing larger building footprints. Resulting in:

  • Clearer layouts
  • Improved player readability
  • More meaningful spatial decision-making

⸻ System Impact

No changes to base grid scale (1 unit = 1 tile maintained). All adjustments performed at object scale level, preserving:

  • Physics consistency
  • Pathfinding stability
  • Existing placement logic

⸻ Summary

This update formalises the transition from prototype-scale placement to a structured, scalable building system, improving both gameplay depth and visual clarity while maintaining core grid integrity.
